What is a Function?

A function is a block of code that performs a specific task. It helps to reuse code, reduce repetition and make programs more organized.

Advantages of Functions

  • ✔ Code reusability
  • ✔ Reduces complexity
  • ✔ Improves readability
  • ✔ Easier debugging

Types of Functions

  • Built-in functions → print(), len()
  • User-defined functions → created using def keyword

Function Syntax

Functions are defined using def keyword followed by function name and parentheses.

Example Programs 💻


def greet():
    print("Hello User")

greet()


def greet(name):
    print("Hello", name)

greet("Vanshu")


def add(a, b):
    return a + b

result = add(5, 3)
print(result)
