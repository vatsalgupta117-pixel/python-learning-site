What is a Variable?

A variable is a named memory location used to store data. In Python, variables are created automatically when you assign a value to them. There is no need to declare the data type explicitly.

Variables make programs dynamic and flexible, allowing data to change during execution.

Rules for Naming Variables:

  • Must start with a letter or underscore
  • Cannot start with a number
  • Cannot use keywords (like if, else, for)
  • Case-sensitive (Name and name are different)

Data Types in Python

Data types define the type of value a variable can store. Python automatically detects the data type.

  • int → Whole numbers (10, 20)
  • float → Decimal numbers (3.14, 2.5)
  • str → Text ("Hello")
  • list → Collection ([1,2,3])
  • bool → True or False

Type Checking

Python provides type() function to check the data type of a variable.

Example Programs 💻
