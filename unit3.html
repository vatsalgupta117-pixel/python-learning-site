What are Control Statements?

Control statements are used to control the flow of execution of a program. They help in decision making and repeating tasks.

Conditional Statements (if-else)

The if-else statement is used to execute code based on conditions. If the condition is true, the "if" block runs, otherwise "else" block runs.

Looping Statements

Loops are used to execute a block of code multiple times.

  • for loop → used when number of iterations is known
  • while loop → used when condition-based repetition is needed

Example Programs 💻


age = 18

if age >= 18:
    print("Eligible to vote")
else:
    print("Not eligible")


for i in range(5):
    print(i)


i = 1

while i <= 5:
    print(i)
    i += 1
