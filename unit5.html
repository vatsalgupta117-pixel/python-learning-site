What is File Handling?

File handling is used to store data permanently in a file instead of temporary memory. It allows programs to read, write, and modify data stored in files.

File Modes

  • "r" → Read file
  • "w" → Write (overwrites file)
  • "a" → Append (adds data)

Why File Handling?

  • ✔ Store data permanently
  • ✔ Useful for real applications
  • ✔ Handles large data efficiently

Example Programs 💻


file = open("data.txt", "w")
file.write("Hello Python")
file.close()


file = open("data.txt", "r")
print(file.read())
file.close()


file = open("data.txt", "a")
file.write("\nNew Line Added")
file.close()
